# README
本文件提供shell下的创建日志的功能
- `log_info` 创建info类型的日志，日志文件为access_shell_2018052922.log
- `log_error` 创建error类型的日志，日志文件为error_shell_2018052922.log

## 使用方法
info类型日志
```
log_info "日志内容。。。"
```
error类型日志
```
log_error "日志内容。。。"
```
